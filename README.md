Description 
This project is a Java-based registration and login system developed using Object-Oriented Programming (OOP) principles.
The application allows a user to:
Register an account using a username, password, and South African cell phone number.
Validate the username to ensure it contains an underscore (_) and is no more than five characters long.
Validate the password to ensure it meets complexity requirements (minimum length, capital letter, number, and special character).
Validate the cell phone number using a regular expression to ensure it includes the international country code (+27).
Log in using the registered username and password.
Receive appropriate success or error messages based on input validation and login status.
The system also includes unit tests to verify that all methods function correctly.
